ABOUT THIS PROJECT

This sports event calendar was built as part of the Sportradar Coding Academy 2026 assessment. It demonstrates backend API development with Spring Boot, PostgreSQL database design with proper normalization, and a clean frontend interface.

The application allows viewing, filtering, and adding sports events with full team and stage information pulled from a live database.

Backend

Spring Boot REST API with JPA, PostgreSQL, and Lombok. Efficient queries with JOIN FETCH to avoid N+1 problems.

Database

PostgreSQL with 3NF normalized schema. Foreign keys prefixed with underscore as per specification.

Frontend

Vanilla HTML/CSS/JS served by Spring Boot. Interactive calendar with event tooltips and add event form.
